# AI Image Generator

This project allows users to generate unique artwork using artificial intelligence. It leverages the power of DALL-E 3 to create stunning visuals from simple text descriptions.

## Key Features

- **Text-to-Image**: Enter any prompt and get matching images instantly.
- **Gallery**: Browse generated images created by the community.
- **Secure**: User accounts and stored history protected by modern authentication.
